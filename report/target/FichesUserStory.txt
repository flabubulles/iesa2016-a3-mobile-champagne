# iesa2016-a3-mobile-champagne
Nom du projet : Champagne

## JOUR 1 
-- Utilisations des éléments :

* Plugin Contact
* Plugin Caméra
* Plugin Géolocalisation


### User Story : Contact
Titre : Demander un renseignement à un de mes contacts

* En tant que : amateur de cocktail, curieux de nouveautés 
* Je veux : demande un avis sur des boissons à l'un de mes contacts.
* Afin de : pouvoir orienter mes achats pour la soirée que je vais bientôt organiser.

Description - Question:
* Contact (API)


### User Story : Camera
Titre : Partager une photo de boisson

* En tant que : photographe amateur
* Je veux : prendre une photo d'une boisson 
* Afin de : la partager pour illustrer un nouveau cocktail

Description - Question:
* Caméra (API)


### User Story : Geolocalisation
Titre : Trouvez les shops les moins cher autour de moi

* En tant que : étudiant sans revenus fixe
* Je veux : retouvez les shop de boissons proposant les meilleurs prix autour de moi
* Afin de : acheter aux meilleurs prix (à spécifier)

Description - Question:
* Géolocalisation (API)
