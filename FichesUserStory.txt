# iesa2016-a3-mobile-champagne
Nom du projet : Champagne

## JOUR 1 
-- Utilisations des éléments :

* Plugin Contact
* Plugin Caméra
* Plugin Géolocalisation


### User Story : Contact
Titre : Demander un renseignement à un de mes contacts

* En tant que : amateur de cocktail, curieux de nouveautés 
* Je veux : demande un avis sur des boissons à l'un de mes contacts.
* Afin de : pouvoir orienter mes achats pour la soirée que je vais bientôt organiser.

Descrition - Question:
* Contact (API)


### User Story : Camera
Titre : Partager une photo de boisson

* En tant que : photographe amateur
* Je veux : prendre une photo d'une boisson 
* Afin de : la partager pour illustrer un nouveau cocktail

Descrition - Question:
* Caméra (API)


### User Story : Geolocalisation
Titre : Trouvez les shops les moins cher autour de moi

* En tant que : étudiant sans revenus fixe
* Je veux : retouvez les shop de boissons proposant les meilleurs prix autour de moi
* Afin de : acheter aux meilleurs prix (à spécifier)

Descrition - Question:
* Géolocalisation (API)


## JOUR 2 
-- Utilisations des éléments :

* Plugin Thème UI
* Plugin Globalization
* Plugin Analytics
* Plugin Accès remote

### User Story : Thème UI
Titre : Naviguer intuitivement

* En tant que : utilisateur de "Champagne"
* Je veux : avoir une interface agréable et simple 
* Afin de : pouvoir me repérer facilement dans l'application / retrouver facilement l'application parmis toutes celles que je possède.

Descrition - Question:
* Thème UI (API)


### User Story : Globalization
Titre : Accèder au site en multilingues

* En tant que : passionné de voyages
* Je veux : que mes contacts à l'étranger accèdent à cette application
* Afin de : pouvoir partager mes boissons préférées testées lors de mes voyages

Descrition - Question:
* Globalization (API)


### User Story : Analytics
Titre : Mesurer la portée des publications

* En tant que : créateur de l'application
* Je veux : mesurer la porté de mon aplication
* Afin de : pouvoir améliorer les points faibles de mon application

Descrition - Question:
* Analytics (API)


### User Story : Accès remote (à revoir)
Titre : Toucher le plus grand nombre de contacts possibles

* En tant que : utilisateur de réseaux sociaux
* Je veux : pouvoir lier l'application à facebook
* Afin de : pouvoir toucher tous mes  contacts et accèder à leurs préférences

Descrition - Question:
* Accès remote (API)
